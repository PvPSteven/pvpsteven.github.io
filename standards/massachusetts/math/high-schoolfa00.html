S-ID Interpreting Categorical and Quantitative Data

  • S-ID.A Summarize, represent, and interpret data on a single count or measurement variable. Use calculators, spreadsheets, and other technology as appropriate.

    • S-ID.A.1 Represent data with plots on the real number line (dot plots, histograms, and box plots).

      • Create bar graphs, line graphs, and histograms (A1-N.2)

    • S-ID.A.2 Use statistics appropriate to the shape of the data distribution to compare center (median, mean) and spread (interquartile range, standard deviation) of two or more different data sets.

      • Mean, median, mode, and range (A1-KK.2)
      • Quartiles (A1-KK.3)
      • Mean absolute deviation (A1-KK.6)
      • Variance and standard deviation (A1-KK.7)
      • Variance and standard deviation (A2-EE.2)
      • Variance and standard deviation (PC-Z.2)

    • S-ID.A.3 Interpret differences in shape, center, and spread in the context of the data sets, accounting for possible effects of extreme data points (outliers).

      • Interpret box-and-whisker plots (A1-N.5)
      • Identify an outlier (A1-KK.4)
      • Identify an outlier and describe the effect of removing it (A1-KK.5)
      • Identify an outlier (A2-EE.3)
      • Identify an outlier and describe the effect of removing it (A2-EE.4)
      • Identify an outlier (PC-Z.3)
      • Identify an outlier and describe the effect of removing it (PC-Z.4)

    • S-ID.A.4 Use the mean and standard deviation of a data set to fit it to a normal distribution and to estimate population percentages. Recognize that there are data sets for which such a procedure is not appropriate. Use calculators, spreadsheets, and tables to estimate areas under the normal curve.

      • Find probabilities using the normal distribution I (A2-DD.11)
      • Find probabilities using the normal distribution (PC-Y.12)
      • Find z-values (PC-Y.13)
      • Find values of normal variables (PC-Y.14)
      • Distributions of sample means (PC-Y.15)
      • The Central Limit Theorem (PC-Y.16)

  • S-ID.B Summarize, represent, and interpret data on two categorical and quantitative variables.

    • S-ID.B.5 Summarize categorical data for two categories in two-way frequency tables. Interpret relative frequencies in the context of the data (including joint, marginal, and conditional relative frequencies). Recognize possible associations and trends in the data.

    • S-ID.B.6 Represent data on two quantitative variables on a scatter plot, and describe how the variables are related.

      • S-ID.B.6.a Fit a linear function to the data and use the fitted function to solve problems in the context of the data. Use functions fitted to data or choose a function suggested by the context. Emphasize linear and exponential models.

        • Find the equation of a regression line (A1-KK.13)
        • Interpret regression lines (A1-KK.14)
        • Analyze a regression line of a data set (A1-KK.15)
        • Find the equation of a regression line (A2-EE.6)
        • Interpret regression lines (A2-EE.7)
        • Analyze a regression line of a data set (A2-EE.8)
        • Find the equation of a regression line (PC-Z.8)
        • Interpret regression lines (PC-Z.9)
        • Analyze a regression line of a data set (PC-Z.10)
        • Analyze a regression line using statistics of a data set (PC-Z.11)

      • S-ID.B.6.b Informally assess the fit of a function by plotting and analyzing residuals.

        • Interpret a scatter plot (A1-KK.8)

      • S-ID.B.6.c Fit a linear function for a scatter plot that suggests a linear association.

        • Scatter plots: line of best fit (A1-KK.12)

  • S-ID.C Interpret linear models.

    • S-ID.C.7 Interpret the slope (rate of change) and the intercept (constant term) of a linear model in the context of the data.

      • Interpret regression lines (A1-KK.14)
      • Analyze a regression line of a data set (A1-KK.15)
      • Interpret regression lines (A2-EE.7)
      • Analyze a regression line of a data set (A2-EE.8)
      • Interpret regression lines (PC-Z.9)
      • Analyze a regression line using statistics of a data set (PC-Z.11)

    • S-ID.C.8 Compute (using technology) and interpret the correlation coefficient of a linear fit.

      • Match correlation coefficients to scatter plots (A1-KK.10)
      • Calculate correlation coefficients (A1-KK.11)
      • Match correlation coefficients to scatter plots (PC-Z.6)
      • Calculate correlation coefficients (PC-Z.7)

    • S-ID.C.9 Distinguish between correlation and causation.

S-IC Making Inferences and Justifying Conclusions

  • S-IC.A Understand and evaluate random processes underlying statistical experiments. Use calculators, spreadsheets, and other technology as appropriate.

    • S-IC.A.1 Understand statistics as a process for making inferences about population parameters based on a random sample from that population.

      • Identify biased samples (A1-KK.1)
      • Identify biased samples (A2-EE.1)
      • Identify biased samples (PC-Z.1)

    • S-IC.A.2 Decide if a specified model is consistent with results from a given data-generating process (e.g., using simulation).

  • S-IC.B Make inferences and justify conclusions from sample surveys, experiments, and observational studies.

    • S-IC.B.3 Recognize the purposes of and differences among sample surveys, experiments, and observational studies; explain how randomization relates to each.

      • Experiment design (A2-EE.12)
      • Experiment design (PC-Z.15)

    • S-IC.B.4 Use data from a sample survey to estimate a population mean or proportion; develop a margin of error through the use of simulation models for random sampling.

      • Find confidence intervals for population means (A2-EE.9)
      • Find confidence intervals for population proportions (A2-EE.10)
      • Find confidence intervals for population means (PC-Z.12)
      • Find confidence intervals for population proportions (PC-Z.13)

    • S-IC.B.5 Use data from a randomized experiment to compare two treatments; use simulations to decide if differences between parameters are significant.

      • Analyze the results of an experiment using simulations (PC-Z.16)

    • S-IC.B.6 Evaluate reports based on data.

      • Interpret confidence intervals for population means (A2-EE.11)
      • Analyze the results of an experiment using simulations (A2-EE.13)
      • Interpret confidence intervals for population means (PC-Z.14)
      • Analyze the results of an experiment using simulations (PC-Z.16)

S-CP Conditional Probability and the Rules of Probability

  • S-CP.A Understand independence and conditional probability and use them to interpret data from simulations or experiments.

    • S-CP.A.1 Describe events as subsets of a sample space (the set of outcomes) using characteristics (or categories) of the outcomes, or as unions, intersections, or complements of other events ("or," "and," "not").

      • Theoretical probability (A1-JJ.1)
      • Compound events: find the number of outcomes (A1-JJ.3)
      • Theoretical and experimental probability (G-X.1)
      • Compound events: find the number of outcomes (G-X.2)
      • Calculate probabilities of events (A2-CC.2)
      • Calculate probabilities of events (PC-X.2)

    • S-CP.A.2 Understand that two events A and B are independent if the probability of A and B occurring together is the product of their probabilities, and use this characterization to determine if they are independent.

      • Identify independent and dependent events (A1-JJ.4)
      • Probability of independent and dependent events (A1-JJ.5)
      • Independent and dependent events (G-X.3)
      • Identify independent events (A2-CC.7)
      • Probability of independent and dependent events (A2-CC.8)
      • Identify independent events (PC-X.6)

    • S-CP.A.3 Understand the conditional probability of A given B as P(A and B)/P(B), and interpret independence of A and B as saying that the conditional probability of A given B is the same as the probability of A, and the conditional probability of B given A is the same as the probability of B.

      • Find conditional probabilities (A2-CC.9)
      • Independence and conditional probability (A2-CC.10)
      • Find conditional probabilities (PC-X.7)
      • Independence and conditional probability (PC-X.8)

    • S-CP.A.4 Construct and interpret two-way frequency tables of data when two categories are associated with each object being classified. Use the two-way table as a sample space to decide if events are independent and to approximate conditional probabilities.

      • Find probabilities using two-way frequency tables (A2-CC.6)
      • Find conditional probabilities using two-way frequency tables (A2-CC.11)
      • Find probabilities using two-way frequency tables (PC-X.5)
      • Find conditional probabilities using two-way frequency tables (PC-X.9)

    • S-CP.A.5 Recognize and explain the concepts of conditional probability and independence in everyday language and everyday situations.

  • S-CP.B Use the rules of probability to compute probabilities of compound events in a uniform probability model.

    • S-CP.B.6 Find the conditional probability of A given B as the fraction of B's outcomes that also belong to A, and interpret the answer in terms of the model.

      • Geometric probability (G-X.7)
      • Find conditional probabilities (A2-CC.9)
      • Find conditional probabilities using two-way frequency tables (A2-CC.11)
      • Find conditional probabilities (PC-X.7)
      • Find conditional probabilities using two-way frequency tables (PC-X.9)

    • S-CP.B.7 Apply the Addition Rule, P(A or B) = P(A) + P(B) – P(A and B), and interpret the answer in terms of the model.

      • Find probabilities using the addition rule (A2-CC.12)
      • Find probabilities using the addition rule (PC-X.10)

    • S-CP.B.8 Apply the general Multiplication Rule in a uniform probability model, P(A and B) = P(A)P(B|A) = P(B)P(A|B), and interpret the answer in terms of the model.

      • Find conditional probabilities (A2-CC.9)
      • Find conditional probabilities (PC-X.7)

    • S-CP.B.9 Use permutations and combinations to compute probabilities of compound events and solve problems.

      • Permutations (A1-JJ.6)
      • Counting principle (A1-JJ.7)
      • Permutation and combination notation (A1-JJ.8)
      • Counting principle (G-X.4)
      • Permutations (G-X.5)
      • Permutation and combination notation (G-X.6)
      • Counting principle (A2-CC.3)
      • Combinations and permutations (A2-CC.4)
      • Find probabilities using combinations and permutations (A2-CC.5)
      • Combinations and permutations (PC-X.3)
      • Find probabilities using combinations and permutations (PC-X.4)
